Provider Migration to Firebase Auth

Migrate existing providers from Firestore-only auth to Firebase Authentication

⚠️ Important Notes

This tool migrates existing providers to Firebase Authentication.
• Only run this once per provider
• Providers will need to use their existing password to log in after migration
• Migration creates a Firebase Auth account linked to their Firestore registration
• This is a one-way migration (cannot be reversed easily)
