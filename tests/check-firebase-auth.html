🔍 Firebase Auth Status Checker

How to Check Firebase Auth Status

  1. Open Browser Console: Press F12 or Ctrl+Shift+I (Windows) / Cmd+Option+I (Mac)
  2. Go to Console Tab: Click on the "Console" tab in DevTools
  3. Click the button below to check your Firebase Auth status automatically
  4. Or copy/paste the code from the code block below into the console

Manual Console Method

Copy and paste this code into the browser console:

// Check Firebase Auth status
import('https://www.gstatic.com/firebasejs/12.3.0/firebase-auth.js').then(({ getAuth }) => {
    const auth = getAuth();
    if (auth.currentUser) {
        console.log('✅ Firebase Auth User:', auth.currentUser.email);
        console.log('✅ Firebase Auth UID:', auth.currentUser.uid);
    } else {
        console.log('❌ Not authenticated with Firebase Auth');
        console.log('💡 You need Firebase Auth, not just Firestore session');
    }
});
